# JSON小工具 - 免费部署方案指南

## 🎯 国内友好免费部署方案

由于GitHub在国内访问不稳定，以下是几个更适合国内环境的免费部署方案：

## 方案一：腾讯云云开发 CloudBase（推荐）

### 优势
- ✅ 国内服务器，访问速度快
- ✅ 完全免费额度充足
- ✅ 支持自定义域名
- ✅ 自动SSL证书
- ✅ 无需服务器维护

### 部署步骤

#### 1. 注册腾讯云账号
- 访问：https://cloud.tencent.com/
- 完成实名认证（免费）

#### 2. 开通云开发服务
1. 进入云开发控制台
2. 创建新环境（选择免费套餐）
3. 记下环境ID

#### 3. 安装云开发 CLI
```bash
npm install -g @cloudbase/cli
```

#### 4. 登录并部署
```bash
# 登录
tcb login

# 部署到云开发
tcb framework deploy
```

#### 5. 绑定自定义域名（可选）
- 在云开发控制台配置域名
- 在域名服务商添加CNAME记录

### 访问地址
- 默认地址：`https://您的环境ID.service.tcloudbase.com`
- 自定义域名：`https://您的域名`

### 免费额度
- 资源使用量：2GB/月
- 函数调用：10万次/月
- CDN流量：5GB/月
- **完全够用，永久免费**

## 方案二：Vercel（国际但国内访问良好）

### 优势
- ✅ 全球CDN，国内访问速度快
- ✅ 支持自定义域名
- ✅ 自动化部署
- ✅ 支持Python应用

### 部署步骤

#### 1. 注册Vercel账号
- 访问：https://vercel.com/
- 使用GitHub账号登录

#### 2. 导入GitHub仓库
1. 在Vercel点击"New Project"
2. 导入您的GitHub仓库
3. 配置部署设置

#### 3. 自动部署
- 每次代码推送到GitHub自动部署
- 无需手动操作

### 访问地址
- 默认：`https://您的项目名.vercel.app`
- 自定义域名：免费支持

### 免费额度
- 带宽：100GB/月
- 函数执行：100小时/月
- **个人项目完全免费**

## 💡 部署技巧

### 提高国内访问速度
1. **启用Gzip压缩**：减少文件传输大小
2. **使用CDN加速**：特别是静态资源
3. **优化图片**：使用WebP格式
4. **减少第三方依赖**：避免加载国外资源

### SEO优化
1. **添加meta标签**：完善页面描述
2. **生成sitemap**：帮助搜索引擎索引
3. **使用语义化HTML**：提高可读性
4. **确保移动端友好**：响应式设计
