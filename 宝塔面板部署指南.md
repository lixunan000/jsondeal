# 宝塔面板部署指南 - 最简单方案

## 🎯 方案概述
使用国内云服务器的免费试用 + 宝塔面板，实现一键部署。

## 📋 准备工作

### 1. 申请免费云服务器（任选其一）

**阿里云**（推荐）：
- 免费试用1个月ECS
- 1核2G配置，完全够用
- 申请地址：https://free.aliyun.com

**腾讯云**：
- 免费试用云服务器
- 配置：1核1G
- 申请地址：https://cloud.tencent.com/free

**华为云**：
- 免费试用弹性云服务器
- 配置：1核2G
- 申请地址：https://activity.huaweicloud.com/free_test

### 2. 安装宝塔面板

在云服务器上执行以下命令（CentOS系统）：
```bash
# 安装宝塔面板
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

# 或者使用国内镜像（更快）
curl -sSO http://download.bt.cn/install/install_panel.sh && bash install_panel.sh
```

安装完成后，记下面板地址、用户名和密码。

## 🚀 部署步骤

### 步骤1：登录宝塔面板
1. 在浏览器打开宝塔面板地址（如：http://服务器IP:8888）
2. 使用安装时显示的用户名密码登录

### 步骤2：安装Python项目管理器
1. 在宝塔面板左侧点击"软件商店"
2. 搜索"Python项目管理器"
3. 点击安装

### 步骤3：创建Python项目
1. 打开"Python项目管理器"
2. 点击"添加项目"
3. 填写项目信息：
   - 项目名称：`json-tools`
   - 路径：`/www/wwwroot/json-tools`
   - Python版本：选择3.7+
   - 框架：Flask
   - 启动方式：`app.py`
   - 端口：7777

### 步骤4：上传代码
1. 在宝塔面板左侧点击"文件"
2. 进入 `/www/wwwroot/json-tools` 目录
3. 点击"上传"，选择您的项目所有文件
4. 等待上传完成

### 步骤5：安装依赖
1. 在Python项目管理器中点击您的项目
2. 点击"模块"标签
3. 点击"安装模块"
4. 分别安装：
   - Flask
   - Flask-CORS

### 步骤6：启动项目
1. 在Python项目管理器中点击"启动"
2. 项目状态显示"运行中"即表示成功

## 🌐 访问您的应用

## 🛠️ 常见问题

A: 检查云服务器安全组是否开放7777端口

### Q: 依赖安装失败？

```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple flask flask-cors
```
