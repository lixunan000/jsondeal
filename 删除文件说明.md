# 已删除的后端文件

由于项目已改为纯前端工程，以下后端文件已被删除：

## 删除的文件列表

- `app.py` - Flask主应用文件
- `wsgi.py` - WSGI入口文件
- `gunicorn_config.py` - Gunicorn配置文件

- `requirements.txt` - Python依赖包列表

### 3. 云服务配置
- `cloudbaserc.json` - 腾讯云云开发配置

## 功能特性

✅ **纯前端实现** - 无需服务器支持
✅ **离线可用** - 所有功能在浏览器中运行
✅ **JSON格式化** - 前端直接格式化JSON
✅ **JSON对比** - 前端实现差异查找
✅ **响应式设计** - 支持移动端
✅ **中文字符支持** - 完美支持中文

## 部署方式

现在项目可以部署到任何静态网站托管服务：

1. **GitHub Pages** - 最简单，完全免费
2. **Vercel** - 全球CDN，部署简单
3. **Netlify** - 拖拽部署，功能丰富
4. **腾讯云静态网站** - 国内访问快

项目现在完全独立，无需任何后端支持！
